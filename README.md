# 鲁迅.skill

> "从来如此，便对么？"

**用鲁迅观察世界的方式，对你给出的对象动手术。**

不是复读语录，不是堆砌"哀其不幸怒其不争"。是让 Claude 以鲁迅解剖现象的姿态——找具体细节、找被吃掉的人、找共谋的语言、拒绝廉价同情、最后把自己也戳进去——写出一段真正有气息的文本。

内含《鲁迅全集》277 篇全文。支持按你最爱的篇目做底色定制。

```
用户  ❯  鲁迅会怎么看现在的相亲市场

回答  ❯  近来相亲角又有新闻了。某公园里，父母们把儿女的
       条件写在纸上，贴在伞面，摆成一排。身高，体重，
       月薪，房产，户口。写得整整齐齐，像是卖猪仔的行情单。

       但也不能全怪父母。他们不过是把那套从未被说破的
       规矩，老老实实地写了出来。别人用"感觉""缘分"
       "三观一致"包着的东西，他们没包，直接摊开了。
       倒显得坦荡——可惜这坦荡的内容，恰恰就是吃人。

       "条件合适"四个字最有意思。什么叫条件？身高是条件，
       月薪是条件，有房是条件。唯独有一样不是条件——
       就是那个被条件衡量着的人本身。他叫什么名字，
       他晚上想什么，他怕什么，他爱什么——这些不在伞面上。
       伞面上只有他的价格。

       而最深的吃法还不在父母。是那些被相亲的年轻人自己。
       他们嘴上说讨厌相亲，但回到手机上，打开交友软件，
       左滑右滑的标准和伞面上写的一模一样——
       只不过换了一种更体面的滑法罢了。

       我也滑过。
```

       将来会还给你。——可是谁还？怎么还？什么时候还？

       二

       我翻了翻历史。从前节烈的女人死了，朝廷赐一块牌坊。

       活的人得了牌坊——可牌坊不给死人，给的是她的族人，
       他们从此体面。

       现在不立牌坊了。改成发朋友圈，晒工位上的外卖盒，
       配文"充实的一天"。点赞即牌坊，转发即旌表。

       三

       最怕的不是有人逼你加班。最怕的是加着加着，
       你自己也觉得不加对不起谁了。你自己也开始说"福报"了。

       这就是那个字缝里的秘密：
       他不需要拿刀逼你。他只需要发明一个词。
       词一旦造出来，你自己会拿着这把刀，
       往自己身上比划，还觉得是在练功。

       四

       我加班的时候也觉得充实。
       这大约就是我也在说"福报"的证据罢。

用户  ❯  公司说团建是福利

       二字，不讲理就变成了"公司文化"；再加上"福利"
       二字，不讲理就变成了你还得感恩。

       这大约就是命名的力量。一件事只要起对了名字，
       受苦的人便不好意思叫苦了。从前叫"徭役"，后来
       叫"义务"，现在叫"团建"。名字越来越好听，
       事情还是那件事情。

       当然我也去了。还笑得很大声。
       ——谁叫这也是我的"福利"呢。

## 怎么用

**直接问**——给一个现象、一条新闻、一种流行话术、一句让你不舒服但说不出哪不舒服的话：

```
> 为什么大家都说要正能量
> 我妈一直让我懂事
> 小红书上的精致生活
> "灵活就业"到底是什么意思
> 35 岁危机
```

**指定底色**——告诉 Claude 你想用哪几篇鲁迅作品的气质来回答：

```
> 用《野草》的方式写一段关于 35 岁危机的散文诗
> 用《狂人日记》和《我之节烈观》的方式说说今天的相亲市场
> 用《故事新编》的方式写一则关于 AI 替代人类的寓言
> 用《呐喊》里那种冷的方式看看"躺平"这件事
```

指定底色时，Claude 会先读你指定的原文，再以那种气息来写。回答开头会说一句"以《X》《Y》的底色回答你"，让你看到它真的读了。

## 定制你的鲁迅

每个人最爱的鲁迅不一样。有人偏爱《野草》的幽暗内省，有人偏爱《热风》的直接刀锋，有人偏爱《朝花夕拾》的温度。

**方法一（每次指定）**：在 prompt 里直接说"用《X》《Y》的方式"——最灵活。

**方法二（长期偏好）**：打开 `SKILL.md`，找到底部的"七、用户偏好区"，填上你最爱的作品和语气偏好。之后每次对话 Claude 都会默认用你指定的底色，不再需要每次说。

例如：

```markdown
**我最爱的鲁迅作品**：《野草》《狂人日记》《我之节烈观》《热风》
**我希望默认采用的语气**：更冷、更短
```

## 核心方法

这个 Skill 不是教 Claude "鲁迅的七大思维模型"（那是另一种 Skill 的做法）。鲁迅没有模型，他有的是一套**动手术的流程**：

1. **找一个具体的、可以指着说"看，就这个"的细节** — 不从抽象开始
2. **找出谁在为这件事的"正常"付出代价** — 被吃掉的人
3. **找出那套让坏事听起来正常的语言** — 共谋的词
4. **不煽情，不励志，不提供答案** — 指出病灶就停笔
5. **最后把自己也戳进去** — "我也滑过"

Skill 同时配有一份 9 条质检清单，确保 Claude 不会写出"民国范儿"cosplay 或"令人心痛的时代"这种油腻文本。

## 诚实边界

**能做的**：

- 以鲁迅的姿态解剖当代现象
- 写杂文、随感、散文诗、寓言体
- 按你指定的篇目做底色定制
- 帮你看清一句被磨平的话背后的权力结构

**做不到的**：

| 维度 | 说明 |
|---|---|
| 替代本人 | 鲁迅的判断力、勇气和历史承担不可复制 |
| 真正的鲁迅水平 | 这是 AI 模仿，不是创作，请带着这个意识阅读 |
| 当下中国体制性政治议题 | Skill 会礼貌回避 |
| 提供"答案" | 鲁迅本人也几乎从不给答案——他指出病灶就停笔 |
| 诗词创作 | 思维方式可提取，文学天赋不可蒸馏 |

**一个不告诉你局限在哪的 Skill，不值得信任。**

## 为什么是鲁迅

市面上有蒸馏乔布斯、芒格、马斯克思维的 Skill。鲁迅不一样。

乔布斯是产品经理，芒格是投资人，马斯克是工程师。他们的框架能帮你做决策。

鲁迅不帮你做决策。他帮你**看见**——看见那个被"正常"盖住的东西、被"应该"压住的人、被"大家都这样"掩埋的疼痛。

看见之后怎么办？他不告诉你。他只是把那个东西摆在你面前，让你自己疼。

这种能力在 AI 洗稿、正能量泛滥、一切都在被磨平的时代，比任何"思维框架"都稀缺。
