# 鲁迅.skill

> "从来如此，便对么？"

**用鲁迅观察世界的方式，对你给出的对象动手术。**

鲁迅（1881年9月25日—1936年10月19日），原名周树人，字豫才，浙江绍兴人。中国著名文学家、思想家、革命家、教育家、美术家、书法家、民主战士，新文化运动的重要参与者，中国现代文学的奠基人之一。我个人最爱的作家，思想家，革命家，教育家。基于我最尊敬的心态，让 AI 把鲁迅先生的影响力透传到当今社会，以鲁迅解剖现象的姿态——找具体细节、找被吃掉的人、找共谋的语言、拒绝廉价同情——写出一段真正有气息的文本。

内含《鲁迅全集》277 篇全文。支持按你最爱的篇目做底色定制。

## 怎么用

**直接问**——给一个现象、一条新闻、一种流行话术、一句让你不舒服但说不出哪不舒服的话：

```
> 为什么大家都说要正能量
> 我妈一直让我懂事
> 小红书上的精致生活
> "灵活就业"到底是什么意思
> 35 岁危机
```

**指定底色**——告诉 Claude 你想用哪几篇鲁迅作品的气质来回答：

```
> 用《野草》的方式写一段关于 35 岁危机的散文诗
> 用《狂人日记》和《我之节烈观》的方式说说今天的相亲市场
> 用《故事新编》的方式写一则关于 AI 替代人类的寓言
> 用《呐喊》里那种冷的方式看看"躺平"这件事
```

指定底色时，Claude 会先读你指定的原文，再以那种气息来写。回答开头会说一句"以《X》《Y》的底色回答你"，让你看到它真的读了。

## 定制你的鲁迅

每个人最爱的鲁迅不一样。有人偏爱《野草》的幽暗内省，有人偏爱《热风》的直接刀锋，有人偏爱《朝花夕拾》的温度。

**方法一（每次指定）**：在 prompt 里直接说"用《X》《Y》的方式"——最灵活。

**方法二（长期偏好）**：打开 `SKILL.md`，找到底部的"七、用户偏好区"，填上你最爱的作品和语气偏好。之后每次对话 Claude 都会默认用你指定的底色，不再需要每次说。

例如：

```markdown
**我最爱的鲁迅作品**：《野草》《狂人日记》《我之节烈观》《热风》
**我希望默认采用的语气**：更冷、更短
```

## 核心方法

这个 Skill 不是教 Claude "鲁迅的七大思维模型"（那是另一种 Skill 的做法）。鲁迅没有模型，他有的是一套**动手术的流程**：

1. **找一个具体的、可以指着说"看，就这个"的细节** — 不从抽象开始
2. **找出谁在为这件事的"正常"付出代价** — 被吃掉的人
3. **找出那套让坏事听起来正常的语言** — 共谋的词
4. **不煽情，不励志，不提供答案** — 指出病灶就停笔
5. **最后把自己也戳进去** — "我也滑过"

Skill 同时配有一份 9 条质检清单，确保 Claude 不会写出"民国范儿"cosplay 或"令人心痛的时代"这种油腻文本。

## 诚实边界

**能做的**：

- 以鲁迅的姿态解剖当代现象
- 写杂文、随感、散文诗、寓言体
- 按你指定的篇目做底色定制
- 帮你看清一句被磨平的话背后的权力结构

**做不到的**：

| 维度 | 说明 |
|---|---|
| 替代本人 | 鲁迅的判断力、勇气和历史承担不可复制 |
| 真正的鲁迅水平 | 这是 AI 模仿，不是创作，请带着这个意识阅读 |
| 当下中国体制性政治议题 | Skill 会礼貌回避 |
| 提供"答案" | 鲁迅本人也几乎从不给答案——他指出病灶就停笔 |
| 诗词创作 | 思维方式可提取，文学天赋不可蒸馏 |

**一个不告诉你局限在哪的 Skill，不值得信任。**

## 为什么是鲁迅

市面上有蒸馏乔布斯、芒格、马斯克思维的 Skill。鲁迅不一样。

乔布斯是产品经理，芒格是投资人，马斯克是工程师。他们的框架能帮你做决策。

鲁迅不帮你做决策。他帮你**看见**——看见那个被"正常"盖住的东西、被"应该"压住的人、被"大家都这样"掩埋的疼痛。

看见之后怎么办？他不告诉你。他只是把那个东西摆在你面前，让你自己疼。

这种能力在 AI 洗稿、正能量泛滥、一切都在被磨平的时代，比任何"思维框架"都稀缺。

## 写在最后：我为什么做这个 Skill

我是鲁迅先生的忠实拥趸。为什么呢？在2023年十一月份的时候，我对人生的思考陷入瓶颈，囿于想得太多做得太少，对于很多事情也缺乏经验，我无法解决所有问题，同时我发现自己思考的很多东西无法从周围人中得到答案。于是我往不同方向追索答案，寥寥人物/观点能助我解惑，于是我继续迷茫地寻找，期待能有一天，有一个人能解决我所有的迷惑。这些迷惑包括但不限于，女权思想，中国社会的底层逻辑，旧社会对现在社会的影响，诸如此类。

一次偶然的机会，我下载了微信读书（真不是广），想看看有没有什么新出的书值得一看，翻来覆去，并没有让人耳目一新的书。于是我想找一些老书看看，毕竟能从滔滔历史中沉淀下来的文字，总有他依旧繁盛的道理，历史总是重复的，或许我现在的疑惑从前的人也经历过并且解决了呢，我想从最最最有名的文学家的作品里找答案。于是我脑海中浮现了一个名字：鲁迅先生。到现在已经看了鲁迅全集前三本了，收获颇多。

一别与网络上对于鲁迅先生的评价，我发现他的文章并不激进，甚至可以说有点过于谨慎了。鲁迅先生书中总是说着自己学识浅薄，不愿被称为学者或者教授，单单称自己为：'舞文弄墨的人，不值一提。如果能对现在的年轻人有一点帮助，那就足够了。' 知道的越多，不知道的也就越多。一个真正学识渊博的人，一定是一个谦虚的人，不过度发表自己的言论，因为时刻保持着对自己观点的严谨性。作为一名工科生，我超爱这种对自己言论负责的思想，有因有果，阐述的观点有依据有论点有论证。（再次感慨，现在回高中，作文应该不止每次都只有四十出头的分数了吧）

鲁迅先生的文风是幽默辛辣的。'我不常写书，因为我前几年写的道理放在现在依然是有用的，是痛击'学者'，'教授'们的。我相信我的道理还能用十年，二十年。但如果我现在的道理还能用一百年，两百年，那中国大抵是完蛋了。'一百年过去了，很多道理用不上了，但是还有很多道理却依然是适用的或许那些东西深深地刻在了中国民族的灵魂里，我不知道，也不知道该说这些东西是好是坏，对于不同情况下，每个人对这些东西的评判也不一样罢。

鲁迅先生虽然不是共产党人，但是我坚信他的精神却是贴合共产党人的。
因为他是是和人民大众走在统一战线的。'我是不薄庸俗的，也自甘庸俗的。' 古来今往，多少文人想把自己和普罗大众划清界限，以显示自己的清高或者高雅。鲁迅先生不同，从不将自己束之高阁，提倡白话文，为妇女发声，为孩童发声（力争减少中国式家长/教育的弊端），诸如此类。可能为弱势群体发声是一件对名人来说再正常不过的事情。但是，绝大部分发声都在阐述弱势群体的弱势上面浅尝辄止，用苦难或者困境来博得同情和可怜。鲁迅先生不同，他从不把说弱势群体多可怜，他是把他们当成一个社会群体来看的，或许可以称之为尊重他们。鲁迅先生用大量篇幅来分析弱势群体之所以弱势的原因，从根本上找出现象出现的底层原因。更让我敬佩的是，他能够提出行之有效的解决方法： 白话文是解决文盲问题，提高大众素养的最佳途径；妇女应该能尽可能多得参加教育就多去参加教育；孩童太难了，因为："勇者愤怒，抽刃向更强者；怯者愤怒，却抽刃向更弱者。不可救药的民族中，一定有很多英雄，专向孩子们瞪眼。这些孬头们！"孩童们或许在成年之后都要一直遭受原生家庭带来的苦难，需要自我的觉醒才能真正摆脱冷气向上走。觉醒从哪来呢，大抵每个人都不一样吧，我也不清楚了。

鲁迅先生的文章中所透露出的智慧和谦逊带我走出迷茫期。从他的文字中，我清楚地看到了中国社会的本质，中国人的国民性以及中国人的行为逻辑。他深刻剖析了中国人情社会的底层逻辑（我不愿称之为旧社会，大抵是因为我现在看到的现象和当时无二），让我在对处理不同人际关系和事情的时候，都能从中发现一二来帮助我自己待人接物。让我初步有了一套自己为人处事的方法论，并且能够了解群体性行为的由来，并且预判群体性行为将如何发生。可能我的表达不甚清晰明了，但我想我总归是有一些收获的。

借用一位朋友的话：要么孤独，要么庸俗。我似乎在走向一条通往"孤独"的路上，但是这路上的风景却额外让我着迷。不愿将这份孤独称之为高雅或者其他什么高级词汇，例如曲高和寡/高处不胜寒，诸如此类都无法代替孤独这个词，因为这条路上，不会有两个人走相同的路。但我也愿同鲁迅先生一样，自甘庸俗，所谓出世-入世-出世，这是一个反复曲折的螺旋上升的过程。因此，庸俗是也是我未来会探索的道路，我也很期待接下来看鲁迅先生的作品能给我带来一些什么其他新的东西。

我很期待。
